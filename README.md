# Global Water Technology Map

An interactive web-based map showcasing global water technology projects, research initiatives, and funding sources.

## Features

- 🌍 Interactive global map with project markers
- 💧 Filter projects by technology type, funding range, and research year
- 📊 Real-time statistics dashboard
- 💰 Detailed funding source information
- 🔬 Latest research highlights
- 📱 Responsive design for mobile and desktop

## Technology Types

- Water Purification
- Water Monitoring
- Water Conservation
- Desalination
- Wastewater Treatment
